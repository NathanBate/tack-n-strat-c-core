# Tack & Strat Asset Shortlist (Public Domain / CC0)

This list is pre-filtered for strategy-game-friendly art and music that can be used without attribution requirements (public domain or CC0).

## Graphics

- Kenney Strategy + UI packs (CC0): [https://kenney.nl/assets](https://kenney.nl/assets)
  - Good starters: hex/terrain tiles, icons, UI frames, cursors.
- OpenGameArt CC0 strategy and hex tiles: [https://opengameart.org/art-search-advanced?field_art_tags_tid=strategy&field_art_type_tid\[\]=9&license=CC0](https://opengameart.org/art-search-advanced?field_art_tags_tid=strategy&field_art_type_tid[]=9&license=CC0)
- Public Domain Vectors (icons/background elements): [https://publicdomainvectors.org/](https://publicdomainvectors.org/)

## Music

- FreePD (Kevin MacLeod public domain tracks): [https://freepd.com/](https://freepd.com/)
  - Suggested mood buckets: `Electronic`, `Fantasy`, `World`.
- Musopen public domain recordings: [https://musopen.org/music/](https://musopen.org/music/)
  - Good for ambient turn-based playlists.
- Freesound CC0 ambient loops: [https://freesound.org/search/?q=ambient+loop&f=license:%22Creative+Commons+0%22](https://freesound.org/search/?q=ambient+loop&f=license:%22Creative+Commons+0%22)

## Recommended First Pass (quick win)

1. Pick one Kenney icon pack for UI action icons.
2. Pick one Kenney/OpenGameArt CC0 tile set for terrain polish.
3. Pick 3-5 FreePD ambient tracks and rotate per turn/phase.
4. Track each downloaded file in `assets/ASSET_LICENSES.md` with source URL and license snapshot.
